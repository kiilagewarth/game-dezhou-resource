# game-dezhou-resource

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖桌面游戏相关的各类辅助工具、信息展示页面及静态资源。仓库本身不针对任何特定域名或网站，所有页面均为独立可运行的 HTML 文件。

## 项目简介

本项目旨在提供一个集中存放桌面游戏相关 HTML 页面的空间。这些页面可能包含规则说明、计分工具、随机事件生成器、资料查询等常见功能。所有页面均以纯静态 HTML 编写，无需后端支持，可直接在浏览器中打开使用。

## 目录说明

```
game-dezhou-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的图片、样式、脚本等资源
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或文件对应一个独立页面，按功能或主题组织。
- **assets/**：公共资源文件，供页面引用。建议页面内资源路径使用相对路径。

## 页面归档说明

- 每个页面均为独立完整 HTML，可离线打开使用。
- 页面之间无相互依赖关系，可单独部署或分享。
- 部分页面可能引用外部 CDN 资源（如字体库、图标库），网络环境下可正常加载。
- 页面内容以桌面游戏相关为主，不涉及具体网站或商业服务。

## 维护说明

- 欢迎提交 Issue 报告页面问题或提出改进建议。
- 若需新增页面，请将 HTML 文件放入 `pages/` 目录下，并保持目录结构清晰。
- 请勿在本仓库中存放任何包含具体域名、网址、营销信息或诱导点击内容的页面。
- 所有提交的页面应确保不侵犯第三方权益，不包含敏感或违规内容。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可使用。
3. 也可将 `pages/` 目录部署到任意静态托管服务。
